# Architecture Decision Record (ADR) Schema

ADRs document significant architectural decisions. They are stored in `memory/Knowledge/` (e.g., `memory/Knowledge/adr-001-use-postgresql.md`).

## File Structure

```markdown
# ADR-[Number]: [Title]

## Status
[Proposed | Accepted | Deprecated | Superseded]

## Context
The issue that we are seeing is...
- What is the problem?
- What are the constraints?
- What are the options?

## Decision
We will use [Solution] because...
- Justification for the choice.

## Consequences
- **Positive**: [Benefit]
- **Negative**: [Trade-off]
- **Risks**: [Potential issue]
```
